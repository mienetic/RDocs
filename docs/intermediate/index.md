# ระดับกลาง (Intermediate)

ยินดีต้อนรับสู่ระดับกลาง! 

ผ่านพื้นฐานมาแล้ว ได้เวลาเจาะลึกเรื่องเจ๋งๆ ที่ทำให้ Rust ทรงพลังจริงๆ! 

:::tip คุณเก่งขึ้นแล้ว! 
ถ้าเข้าใจ Ownership แล้ว ส่วนที่เหลือจะง่ายขึ้นเยอะเลย!
:::

## หลังจบระดับนี้คุณจะ...

สร้าง custom data types ได้ 
จัดการ error อย่างมืออาชีพ 
ใช้ collections ได้คล่องแคล่ว 
จัดโครงสร้าง project ขนาดใหญ่ 
เข้าใจ Traits และ Generics 
เขียน network applications ได้ 
ใช้ crates ยอดนิยมเป็น (chrono, serde, regex) 
สร้าง documentation มืออาชีพ
